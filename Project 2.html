Project 2

First 3D Print

I had never 3D printed before so the first print of the keychain was a fun experience for me. I did not really know what I was doing so I was worried that I was doing something wrong. The print started good, but I did not notice when it stopped and needed me to change the filament color, so that added some time to the printing process. Overall this was a fun experience and I am excited to continue printing, but I will be more attentive to the color because, while the colors ended up being ok, there were definitely other colors I would have chosen.

Second 3D Print

In my second print I decided to print a model of Dr. Doom because the overhangs of the hood could serve as a good failure for the machine. Also the shape of the model provided low bed adhesion because the bottom of the model was angled. In order to solve this problem of low bed adhesion I lowered the model further down the y-axis, giving the model the ability to stick to the bed. Further problems arose when the print finished because, upon further inspection the chest of the model was minimally detailed as a result of the print being small. Meaning that I learned that a more detailed model needs to be printed in a larger size when using the Prusa MINI+. Furthermore, the tip of the hood did not fully print because it was too far overhung. This tells me that when the print is smaller the greater the overhang can be achieved while if the print is larger parts with extreme overhang will not print. Lastly, 3MF files give detailed 3D data including color and materials, while STL files provide basic geometric mesh data without additional features.
